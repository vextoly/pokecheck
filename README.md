# Vextoly Pokécheck

Vextoly Pokécheck ist ein minimalistisches Web-Tool zur Analyse von Pokémon-Daten. Es wurde gezielt entwickelt, um Spielern auf **Pokémon Showdown** einen schnellen Informationsvorteil zu verschaffen, ohne den Fokus auf das Match zu verlieren.

## Features

* **Showdown Optimiert:** Schneller Zugriff auf Daten, die im kompetitiven Kampf (UI) oft erst gesucht werden müssen.
* **Zweisprachige Suche:** Volle Unterstützung für **deutsche und englische** Pokémon-Namen sowie Pokédex-Nummern.
* **Typen-Analyse:** Detaillierte Auflistung von Schwächen (2x, 4x), Resistenzen (0.5x, 0.25x) und Immunitäten.
* **Intelligente Suche:** Dank Levenshtein-Algorithmus werden Tippfehler erkannt und Korrekturvorschläge gemacht.
* **Live-Vorschläge:** Während der Eingabe erscheinen automatische Vorschläge für eine schnellere Navigation.
* **Basiswerte (Stats):** Einblendbare Statistiken (KP, Angriff, Vert., etc.) direkt unter der Typen-Tabelle.
* **Fähigkeiten-Check:** Dynamisches Laden von Fähigkeiten über die PokéAPI mit Verlinkung zum Bisafans-Fähigkeitendex.
